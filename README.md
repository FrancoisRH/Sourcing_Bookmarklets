# Sourcing_Bookmarklets
This repository lists Google Chrome bookmarlets I use for sourcing purposes

How to install and use bookmarlets in Google Chrome? 
1. Show the bookmarks bar in Google Chrome if it's hidden (press Ctrl + Shift + B)
2. Right click anywhere on the bookmarks bar → click "Add page…"
3. A small dialog appears with two fields:
- Name: type something like "Scrape URLs" if you're using the script to scrape URLs
- URL: paste the entire script (you will find scripts in the files of this repository)
3. Click Save
